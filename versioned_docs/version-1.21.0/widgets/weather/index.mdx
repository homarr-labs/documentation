The Weather Widget uses your location to display the current, highest and lowest temperature. The Widget uses [open-meteo.com](https://open-meteo.com) to retrieve weather data.

![](./img/screenshot.png)

![](./img/screenshot-forecast.png)

## Adding the widget
Please check out our documentation on [how to add a widget](/docs/getting-started/after-the-installation#embed-integration-data-using-widgets).
