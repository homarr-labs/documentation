[Gluetun](https://github.com/qdm12/gluetun) is a lightweight, OpenVPN- and Wireguard-capable VPN client container with a built-in HTTP control server. Homarr reads that control server to surface your live VPN state — status, public IP, exit location, provider and protocol — in the [Gluetun widget](../../widgets/gluetun).

## Setting up the Gluetun control server

Homarr connects to Gluetun's HTTP control server, which is disabled by default. Enable it and configure authentication by setting the control-server environment variables and mounting an auth config file — see the [Gluetun authentication guide](https://github.com/qdm12/gluetun-wiki/blob/main/setup/advanced/control-server.md#authentication).
